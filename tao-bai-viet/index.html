Giới thiệu tổng quan

Easybuider là một công cụ mạnh mẽ được thiết kế để nâng cao hiệu quả làm việc của bạn:

  • là một ứng dụng giúp chủ website, nhà biên tập và tác giả quản lý website của họ và xuất bản nội dung mà không cần kiến thức lập trình.
  • Một trong số những ưu điểm lớn nhất của Easybuilder là giao diện dễ sử dụng và liên tục. Bạn có thể dễ dàng tạo và xuất bản nội dung chỉ trong vài phút!
  • Thực tế, cũng vì việc này mà đã có rất nhiều người đã tạo ra được những website hiện đại, chất lượng cao, bất kể họ là những người mới vào nghề hay đã có kinh nghiệm lâu năm. Một số dự án thực tế mà có thể các bạn biết đã được tạo bởi Easybuilder như : thuysan247.com (thuộc top 6 trang tin tức về thủy sản trên google search), http://ungthu.club/, http://dentrang.net/ ...

    • Ưu điểm và nhược điểm

    Ưu điểm
    • 1. Chi phí thấp : Bạn chỉ cần thanh toán tên miền và web hosting. Phần mềm Easybuilder, theme đều hoàn toàn miễn phí
    • 2. Dễ cài đặt và cập nhật : Không như những phần mềm khác, Easybuilder không cần bất kỳ cấu hình đặc biệt nào, và bạn có thể cập nhật trong 1 click.
    • 3. Quản trị đơn giản : BBạn không cần biết kiến thức lập trình khi tạo website bằng Easybuilder,Các thao tác của bạn chỉ xoay quanh việc: viết và chỉnh sửa bài viết, upload và sửa hình ảnh, quản lý bài viết, thêm nenu ...
    • 4. Giao diện tùy chỉnh : Với hàng trăm theme sẵn sàng để bạn sử dụng bạn có thể dùng, để tạo một thiết kế riêng phù hợp với yêu cầu , phong cách của bạn VD: sự kiện, thể thao, du lịch ...
    • 5. Hỗ trợ : Mọi thắc mắc, ý kiến, hoặc gặp bất kỳ khó khăn nào, chúng tôi luon sẵn sàng trợ giúp bạn.
    Nhược điểm
    • 1. Kích thước website :Nếu website có quá nhiều bài viết, site có thể bị chậm. (Chúng tôi đã có sẵn giải pháp cho bạn đó là CMS)
    • 2. Với các website xử lý phức tạp như : Webiste web-app (ứng dụng trực tuyến), mạng xã hội, blog ... bạn cần một giải pháp tốt hơn (Chúng tôi đã có sẵn giải pháp cho bạn đó là CMS)

    HƯỚNG DẪN CÀI ĐẶT

    Để tạo được một website bạn cần phải tải 2 phần mềm sau :
    • Easyweb-tool-master (link và video hướng dẫn ở trang chủ): đây là một công cụ để bạn có thể download source của bất kỳ một website nào mà bạn muốn với các thao tác đơn giản và nhanh chóng
    • Easybuilder: Là công cụ để bạn tạo một website cho riêng mình từ theme mà bạn chọn trước đó

    Lưu ý:Công ty không có trách nhiệm cũng như khuyến khích download các trang nếu không có sự đồng ý từ bên webiste bạn chọn

    Yêu cầu:

    • Máy tính (Hoặc PC): Cấu hình RAM (tối thiểu) : 4GB

      Lưu ý: Hiện tại phần mềm chỉ đang hỗ trợ trên hệ điều hành Windows

    • Kiến thức (cơ bản): Phải biết các thẻ cơ bản HTML
    • Kiến thức về máy tính:
    • Biết sử dụng một phần mềm để code VD: visual code, Sublime Text, nodepad ...

    HƯỚNG DẪN THỰC HIỆN

    (Thực hiện tạo một website hoàn chỉnh tử công cụ easybuilder)

    Bước 1: Clone website

    Bạn cần tải phần mềm Easyweb-tool-master

    Các bước thực hiện:

    • Chọn theme: hãy chọn một bất kỳ một website nào mà bạn muốn
    • Tải source:
      • B1 : Giải nén easyweb-tools-master
      • B2 : Edit file get.pat
      • B3 : Paste đường dẫn website mà bạn chọn vào
      • B4 : Nhấn đúp chuột vào file get.pat để tiến hành tải source

      • Lưu ý : Viêc tải có thể mất 1 - 2 phút tùi website, chương trình sẽ tự động đóng khi tải xong không (không được tắt chương trình khi đang chạy)

      • Xem video hướng dẫn tại đây

    Bước 2: Tải easybuilder

    • Chọn DOWNLOAD để thực hiện việc tải công cụ easybuilder về máy của bạn
    • Giải nén và chạy file EasyWebBuilder.exe
    • Đăng nhập
    • Nếu bạn chưa có tài khoản hãy đăng ký vào
      • Bạn phải có một “Email” để có thể đăng ký tài khoản trên nếu chưa có hãy đăng ký “tại đây”
      • Đăng ký trên “easybuilder”

    Bước 2: Tạo một project

    • Để tạo một project :
      • chọn SKELETON
      • Điền tên project
      • Add

    Lưu ý: Đặt tên các project phải khác nhau và các tên có ký tự đặt biệt sẽ không được chấp nhận Sau khi tạo project,Easybuilder sẽ tạo sẵn cho bạn một bộ source nằm trong folder site

    • Tìm hiểu tổ chức file trong project : link

    Bước 3: Tạo webiste từ theme đã chọn

    • Thêm file cần thiết: bạn vào source đã tải bằng easyweb-tool-master copy các file có trong source và paste vào Asset trong project
    • Lấy HTML template đưa vào project bạn vào source đã tải bằng easyweb-tool-master copy các file có trong source và paste vào index.html trong folder Layout của project

      • Lưu ý: Cần điều chỉnh các đường dẫn trong code HTML , do một số template khi download về trong đường dẫn không có "/"

    • Build :để hiện thị website bạn vừa tạo

    Bước 4: Đưa các tập tin cần thiết vào website

    • Sync : Để đồng bộ code giữa máy của bạn và server (cũng có thể hiểu là đưa code từ máy bạn lên server)
    • Deploy : Đưa project bạn vừa tạo hiển thị ra website
    • Thao tác có thể mất 1 - 2 phút phụ thuộc và size của project
